## Giới thiệu / Introduction

### Tiếng Việt

Đây là dự án Next.js của Nội Thất Ao Vua, chuyên thiết kế và thi công nội thất cao cấp. Dự án này nhằm mục đích tạo ra một trang web hiện đại và thân thiện với người dùng để giới thiệu các sản phẩm và dịch vụ của chúng tôi.

### English

This is the Next.js project of Nội Thất Ao Vua, specializing in high-end interior design and construction. This project aims to create a modern and user-friendly website to showcase our products and services.

## Cài đặt / Installation

### Tiếng Việt

1. Sao chép kho lưu trữ:
   ```bash
   git clone https://github.com/NguyenDuyThanhCTU/hoangyennest.git
   ```
2. Cài đặt các gói phụ thuộc:
   ```bash
   cd yourproject
   npm install
   ```
3. Chạy dự án:
   ```bash
   npm run dev
   ```

### English

1. Clone the repository:
   ```bash
   git clone https://github.com/NguyenDuyThanhCTU/hoangyennest.git
   ```
2. Install dependencies:
   ```bash
   cd yourproject
   npm install
   ```
3. Run the project:
   ```bash
   npm run dev
   ```

## Cách sử dụng / Usage

### Tiếng Việt

- Truy cập `http://localhost:3000` trên trình duyệt của bạn để xem trang web.
- Chỉnh sửa mã nguồn trong thư mục `app` và `components` để thay đổi nội dung và giao diện của trang web.

### English

- Visit `http://localhost:3000` in your browser to view the website.
- Edit the source code in the `app` and `components` directories to change the content and appearance of the website.

## Cấu trúc thư mục / Project Structure

### Tiếng Việt

- `app/`: Chứa các trang của ứng dụng.
- `components/`: Chứa các thành phần React được sử dụng trong các trang.
- `public/`: Chứa các tài nguyên tĩnh như hình ảnh và tệp tin.

### English

- `app/`: Contains the application's pages.
- `components/`: Contains React components used in the pages.
- `public/`: Contains static assets such as images and files.
